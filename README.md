# andreeabogdanovici.github.io

Jekyll blog for GitHub Pages.

## Add a post

- Create `_posts/YYYY-MM-DD-title.md`
- Front matter:
  - `layout: post`
  - `title: '...'`

## Add a page

- Create `_pages/name.md`
- Front matter:
  - `layout: page`
  - `title: '...'`
  - `permalink: /name/`
  - `nav_order: 10`

## Local preview (optional)

- `jekyll serve`
